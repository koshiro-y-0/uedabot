## 1. プロジェクト概要

本システムは、日本銀行が公開するAPIおよびその他の無料データソースを活用し、金利・為替・短観・CPIなどの主要経済指標を自動取得・整形して通知するBotです。AI/APIの従量課金を一切使用せず、**完全無料**で運用できる構成を採用します。

### 1.1 目的

- 日銀・総務省の公開データを毎朝自動取得し、主要指標をサマリー形式で配信する
- 植田総裁の口調を模したテンプレート文章で読みやすく整形する
- **コスト：¥0**（無料APIのみ使用、LLM不使用）

## 5. 機能仕様

### 5.1 定期配信（毎朝レポート）

- 平日 09:00（JST）に GitHub Actions が自動起動
- 金利・為替・CPI・短観の最新値を取得
- 植田総裁口調テンプレートに値を埋め込み
- LINE または Discord Webhook に送信

### 5.2 アラート通知

| トリガー条件 | アラート種別 |
|---|---|
| USD/JPY が前日比 ±1.5円以上変動 | 緊急為替アラート |
| 金利が前回から変動 | 政策変更アラート（会合翌日） |
| CPI が前月比 ±0.5%以上 | 物価動向アラート |

### 5.3 テンプレート方式（AIなし）

植田総裁の発言パターンを定型文として事前定義し、取得したデータを差し込む方式を採用します。AIは使用しないため、従量課金は一切発生しません。

| 条件 | テンプレート文例 |
|---|---|
| 金利 変化なし | 「物価の基調は概ね見通し通り。先行きのリスクは上下双方向に存在します。」 |
| 金利 引上げ | 「物価安定目標の持続的実現に向け、緩やかな利上げ軌道を維持します。」 |
| CPI 上昇 | 「賃金と物価の好循環が確認されつつある一方、海外リスクに注意が必要です。」 |
| CPI 低下 | 「物価上昇の基調は維持されていますが、先行き不確実性を注意深く見極めます。」 |

## 10. 制約・注意事項

- Yahoo Finance の非公式APIはアクセス頻度を上げすぎると制限される場合があります（1日1〜2回程度の取得を推奨）
- 日銀IMASのデータは速報性に制限があり、発表直後は反映に遅延が生じる場合があります
- LINE Messaging API の無料枠は月200通のため、1日1通×30日 = 月30通で余裕があります
- テンプレート方式のため、複雑な文脈判断や想定外の指標組み合わせへの対応は手動でテンプレートを追加する必要があります
