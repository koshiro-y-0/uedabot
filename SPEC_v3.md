# 🏦 植田総裁Bot — v3.0 拡張機能仕様書

**Version 3.0 | 2026年3月**
**コスト完全ゼロ構成を維持**

---

## 1. 拡張の目的

v2.0 では Quick Reply による詳細情報取得と毎朝の振り返り・見通し機能を実装した。v3.0 では以下の4つの柱で機能を拡充する：

1. **週間サマリー＋チャート画像配信**（毎週金曜日）
2. **過去データの蓄積・比較機能**
3. **経済用語解説機能＋リッチメニュー**
4. **リアルタイムアラート通知の強化**

---

## 2. 新機能一覧

### 2.1 週間サマリー＋チャート画像（毎週金曜日配信）

毎週金曜日の 17:00 に、今週1週間の経済指標の振り返りをサマリーとチャート画像で配信する。

#### メッセージイメージ

```
📊 【週間サマリー】2026年3月9日〜3月13日
━━━━━━━━━━━━━━━━━━━━
📅 今週の振り返り
━━━━━━━━━━━━━━━━━━━━

💱 為替（USD/JPY）
  週初：151.20円 → 週末：152.80円
  週間変動：+1.60円（+1.06%）
  週間高値：153.50円（3/11）
  週間安値：150.90円（3/10）

🎯 政策金利：0.50%（変更なし）

📈 CPI（最新）：総合 3.2%  コア 2.8%

🏭 短観DI：+12（変更なし）

━━━━━━━━━━━━━━━━━━━━
📰 来週の注目イベント
━━━━━━━━━━━━━━━━━━━━
・3/19（水）日銀金融政策決定会合（結果発表）
・3/21（金）CPI（2月分）発表

━━━━━━━━━━━━━━━━━━━━
「今週も市場は概ね安定的に推移しました。
 来週の政策決定会合の結果に注目が集まります。」
```

※ 上記テキストに加えて、**為替チャート画像**（5日間の折れ線グラフ）を送信

#### チャート画像仕様

- matplotlib で生成する PNG 画像（幅800px × 高さ400px）
- USD/JPY の5日間推移を折れ線グラフで描画
- グリッド付き、日付ラベル付き
- 画像は一時ファイルとして生成 → LINE Image Message で送信
- GitHub Actions 上で matplotlib の `Agg` バックエンドを使用（GUI不要）

### 2.2 過去データの蓄積・比較

毎朝のレポート生成時に、その日の指標データを CSV ファイルに追記保存する。蓄積データは将来の比較機能やグラフ生成に活用する。

#### 保存形式

```
data/daily_indicators.csv
```

| カラム | 型 | 例 |
|---|---|---|
| date | str | 2026-03-14 |
| weekday | str | 金 |
| usdjpy | float | 152.30 |
| eurjpy | float | 163.45 |
| usdjpy_prev | float | 151.80 |
| policy_rate | float | 0.50 |
| cpi_total | float | 3.2 |
| cpi_core | float | 2.8 |
| tankan_di | int | 12 |

#### 保存タイミング

- `main.py` の実行時（毎朝8:30）にデータ取得後、CSV に1行追記
- GitHub Actions で CSV を自動コミット（`git add data/ && git commit && git push`）

#### Quick Reply からの利用

「詳細:比較」ボタンを追加し、直近30日間 or 前月との比較データを返す。

### 2.3 経済用語解説機能

ユーザーが「解説:CPI」のようにメッセージを送ると、初心者向けの経済用語解説を返す。

#### 対応用語

| コマンド | 解説内容 |
|---|---|
| `解説:CPI` | 消費者物価指数とは？総合・コア・コアコアの違い |
| `解説:短観` | 日銀短観とは？DIの読み方・調査時期 |
| `解説:政策金利` | 無担保コール翌日物レートとは？金利変更の影響 |
| `解説:為替` | 円安・円高の意味、為替変動の要因 |
| `解説:GDP` | 国内総生産とは？実質・名目の違い |
| `解説:日銀` | 日本銀行の役割・金融政策決定会合とは |

#### 解説テンプレート例

```
📖 用語解説：CPI（消費者物価指数）
━━━━━━━━━━━━━━━━━━━━

■ CPIとは
消費者が購入する商品・サービスの価格変動を
測る指数です。前年同月比で表示されます。

■ 3つの種類
・総合CPI：すべての品目を含む
・コアCPI：生鮮食品を除く（日銀が重視）
・コアコアCPI：食料・エネルギーを除く

■ なぜ重要？
日銀は「物価安定目標2%」を掲げており、
CPIはその達成度を測る最重要指標です。
CPIの動向は金融政策に直結します。

━━━━━━━━━━━━━━━━━━━━
「物価の安定は、あらゆる経済活動の
 基盤であると考えております。」
```

### 2.4 リッチメニュー

LINE の画面下部に常設メニューを配置する。Quick Reply と異なり、常時表示されるためユーザーが迷わず操作できる。

#### メニュー構成（6分割）

```
┌──────────┬──────────┬──────────┐
│ 📊 為替  │ 🎯 金利  │ 📈 CPI  │
├──────────┼──────────┼──────────┤
│ 🏭 短観  │ 📖 用語  │ 📰 注目  │
└──────────┴──────────┴──────────┘
```

| ボタン | 送信テキスト | 動作 |
|---|---|---|
| 📊 為替 | `詳細:為替` | 為替詳細レポート |
| 🎯 金利 | `詳細:金利` | 金利詳細レポート |
| 📈 CPI | `詳細:CPI` | CPI詳細レポート |
| 🏭 短観 | `詳細:短観` | 短観詳細レポート |
| 📖 用語 | `解説:一覧` | 用語解説メニューを表示 |
| 📰 注目 | `詳細:注目` | 今日の注目イベント |

#### リッチメニュー画像

- 画像サイズ: 2500px × 843px（LINE推奨サイズ）
- Python の Pillow ライブラリで自動生成（`src/generate_richmenu.py`）
- LINE Messaging API でリッチメニューを登録（初回のみ手動実行）

### 2.5 リアルタイムアラート通知の強化

為替の急変動（USD/JPY が短時間で ±2円以上）を検知し、即時通知する。

#### 実装方式

- GitHub Actions の cron を **平日の9:00〜18:00に15分おき**に実行するワークフローを新設
- 直前のレート（CSV蓄積データ）と現在のレートを比較
- 閾値超えの場合のみ LINE Push Message を送信（通常時は何もしない）

#### アラートメッセージ

```
⚠️ 【為替アラート】2026年3月14日 14:30
━━━━━━━━━━━━━━━━━━━━
USD/JPY が急変しました！

  前回チェック時：150.50円（14:15）
  現在値：152.80円
  変動幅：+2.30円（+1.53%）

📊 トレンド：急激な円安方向
「為替市場の急変に際しては、
 適切に対応してまいります。」
━━━━━━━━━━━━━━━━━━━━
```

#### ワークフロー設定

```yaml
# .github/workflows/forex_alert.yml
on:
  schedule:
    # 平日 09:00-18:00 JST に15分おき（UTC 00:00-09:00）
    - cron: "*/15 0-9 * * 1-5"
```

#### 注意事項

- GitHub Actions の無料枠は月2,000分。15分おき × 10時間 × 20日 = 月800回。1回あたり30秒程度 → 月400分程度。無料枠内で収まる。
- 為替の閾値は ±2円（デフォルト）。環境変数 `FOREX_ALERT_THRESHOLD` で変更可能。

## 3. システムアーキテクチャ（v3.0）

```
[毎朝 08:30 定期配信]
GitHub Actions (cron 月〜金)
    ↓
fetch → generate → notify (Quick Reply + リッチメニュー)
    ↓
CSV にデータ蓄積 → git commit & push
    ↓
LINE Push Message（振り返り＋見通し付き）

[毎週金曜 17:00 週間サマリー]
GitHub Actions (cron 金曜のみ)
    ↓
CSV から週間データ集計 → matplotlib チャート生成
    ↓
LINE Push Message（テキスト＋チャート画像）

[平日 15分おき 為替アラート]
GitHub Actions (cron 09:00-18:00)
    ↓
為替レート取得 → CSV の直前値と比較
    ↓ 閾値超え時のみ
LINE Push Message（アラート通知）

[インタラクティブ応答]
LINE Webhook → Vercel (serverless)
    ↓
詳細情報 / 用語解説 / 比較データ を Reply
```

## 9. 制約・注意事項

- GitHub Actions の cron は正確なタイミングでは発火しない（数分〜十数分の遅延あり）。為替アラートは「概ね15分おき」程度の精度。
- LINE Push Message は月200通まで無料。毎朝配信（約22通）＋週間サマリー（4通）＋アラート（想定月10通程度）= 合計36通程度。十分余裕あり。
- CSV ファイルが大きくなりすぎた場合（数年後）、古いデータのアーカイブを検討する。
- リッチメニュー画像は一度作成・登録すれば変更するまでそのまま使用できる。
- matplotlib のフォント設定（日本語表示）に注意。GitHub Actions の Ubuntu 環境では `Noto Sans CJK` を使用する。
